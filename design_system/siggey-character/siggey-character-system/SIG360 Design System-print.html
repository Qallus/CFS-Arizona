Visual Principles
Visual Principles

Six rules that hold the system together.

A
Editorial restraint
Generous whitespace, hairline dividers, and a serif voice at brand moments. Density only where data demands it.
B
Navy as foundation
Navy carries weight and authority. Ivory carries calm. Gold is reserved for moments that matter.
C
Quiet intelligence
AI shows up as a soft glow, a careful suggestion, a precise summary — never as fireworks.
D
Operational clarity
Status, ownership, and next action are visible in one glance. The UI helps the user decide, then commit.
E
Refined geometry
Rounded but not soft. 12 / 16 / 20px radii, thin 1px borders, gentle elevation. Nothing oversized or trendy.
F
Human warmth
Warm ivory, soft tan, and the Siggey character keep a deeply operational product feeling humane.

Rules
Body text minimum 4.5:1. Interactive elements minimum 3:1 for non‑text. Never pair gold on ivory for body copy — only icons and accents.
Focus rings
Royal blue ring at 2px outside, 2px offset. Increase to 3px in dark mode. Never rely on color alone — always add an outline or shape change.
Status without color
Every status uses an icon and a label. Color is a second signal — colorblind users get the full message from the text alone.

Mode Rules
Mode rules

Two atmospheres, same standards.

Light mode · default
Warm office, clearly lit.
Warm ivory background. Navy text. White and soft tan surfaces. Gold for honor and primary actions. Hairline borders. Calm, financial‑planning aesthetic.
Use when · daytime planning sessions, advisor desk work, printable reports.
Gold use · primary CTA per surface, brand moments, AI highlight.
Navy use · primary text, nav bars, dense data headers.
Dark mode · command center
Quiet command, after hours.
Deep navy foundation. Slightly lighter navy surfaces. Gold accents. Muted ivory text. Soft blue borders. Subtle AI glow where intelligence shows up.
Use when · operational reviews, automation building, late‑session triage.
Gold use · primary CTA, brand mark, AI subject lines.
Ivory use · primary text, headings, navigation.
Keep both modes premium · do‑not list
✕ Never use pure white in dark mode for body text.
✕ Never stack three glows on one screen.
✕ Never use gold as a background fill.
✕ Never reverse the wordmark with shadows.

Usage rules & animation
Usage & animation

A guide, never a mascot.

Do & don't
✓
Use Siggey when there's something to say — a suggestion, a status, a guide.
✓
Pair Siggey with calm, plain‑language copy. Italic serif for the line itself.
✓
Show Siggey at most once per surface — be selective.
✓
Match the expression to the moment — never default smile for compliance.
✕
Don't recolor Siggey outside the SIG palette.
✕
Don't put Siggey in adult clothing, costumes, or seasonal accessories.
✕
Don't animate continuously — motion is reserved for state changes.
✕
Don't use Siggey as a marketing celebrity outside the product voice.
Animation direction
Idle breath
Scale 1.0 → 1.015 → 1.0 over 3.2s. Inner light ±10%.
Thinking pulse
Eyes look up + thought bubble fades in, 1.4s loop.
Loading sweep
Gold dashed ring rotates 1.2s, eyes closed.
Success bloom
Eyes ^_^, smile widens, sparkles fade in for 480ms.
Wave
Left stub arm rotates ±15° twice, 600ms. Greetings only.

10 · Motion
Principles & durations
Motion principles

Four principles. Three durations. One curve.

A
Purposeful
Every animation reveals a state change. If nothing changed, nothing should move.
B
Calm
No bounce, no spring overshoot. Smooth eases that finish quietly.
C
Brief
120–320ms covers 95% of needs. Anything longer should be a Siggey moment.
D
Considerate
Respect prefers-reduced-motion. Replace transitions with instant fades.
Durations & easings
--motion-instant0 ms · for reduced motion
--motion-fast120 ms · hover, focus
--motion-base200 ms · most state changes
--motion-slow320 ms · drawers, modals enter
--motion-siggey480–800 ms · Siggey moments only

Sentence pattern
Subject · plain verb · object · optional next action. One idea per sentence.
Words we use

Words we avoid
